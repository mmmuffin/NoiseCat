Chrome Extension

NoiseCat

用共享屏蔽词库快速屏蔽大家都讨厌的广告评论，并可按需批量写入你自己的 X 屏蔽词。

插件开关

关闭后，划词添加、公共词库隐藏和批量同步都会停用。

公共词库

填写 GitHub raw 文件地址，例如 `https://raw.githubusercontent.com/mmmuffin/NoiseCat/main/community-keywords/keywords.json`。

当前行为

启用公共词库后，插件会按下面两种方式工作。

  1. 在推文详情页中，命中公共词库的评论会被直接隐藏。
  2. 你可以点击“同步到 X 屏蔽词”，把公共词库批量写入自己的 X 设置。
  3. 划词添加单个屏蔽词的功能会继续保留。

最近一次结果

  • 暂无记录
  • 需要你在 Chrome 里已经登录 X。
  • 当前公共词库拉取仅支持 `raw.githubusercontent.com`。
